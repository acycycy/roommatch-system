근데 룸메이트 매칭을 하려면 마일리지라는 걸 모으고, 따로 개인이 룸메이트를 구해서 학교에 신청을 해야함
사람들이 에브리타암에다가 자기 생활패턴이나 원하는 룸메이트 조건 적어서 연락달라고 항상 학기초에 계속 올림
이걸 보면서 든 생각이 어차피 신뢰도는 에타에서 구하는 방식이나 이거나 별반 다르지 않으니 양측에서 설문을 받고 맞는 사람을 룰기반으로 매칭해주는 매칭 프로그램?을 만들어서 매칭을 시켜주고 어플이나 메일로 알림을 가게 해주는건 어떨까 생각함
일단 조건들을 모아봄 (선호 수면시간대, 흡연여부, 시험기간 소등여부, 청소스타일 등등)
기본적으로는 룰기반 매칭을 할건데 진짜 ai도 사용해서 사용자들끼리 매칭추천도 해주려고 함
클러스터링 + 임베딩으로 사용자 유사도 기반 매칭까지 고려중
초기엔 더미데이터로
하지만 무작위면 이상한 결과가 나올 수도 있으니까 유형을 정해놓음
타입 A: 아침형 + 조용함 + 청결 중시 취침시간 22~23시 기상시간 6~7시 소음 민감도 높음 청소 자주 함 대화 선호 낮음 
타입 B: 야행성 + 생활 자유로움 취침시간 1~3시 기상시간 9~11시 소음 민감도 낮음 청소 적당 야간활동 많음
