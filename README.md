# roommatch-system

Rule-based roommate matching system with conflict-aware scoring and hard cutoff logic

생활 패턴과 갈등 요소를 반영하여 룸메이트를 매칭하는 시스템


## 기존 룸메이트 매칭 서비스는
단순 조건 필터링 방식으로 실제 생활 갈등을 충분히 반영x


## 
- 항목별 가중치 기반 점수 계산
- 갈등 요소(수면시간 등) 높은 가중치 부여
- 하드 컷오프 적용
  - 예: 수면시간 차이 4시간 이상 → 최대 점수 제한


## 
- 사용자 설문 기반 데이터 수집
- 룰 기반 매칭 알고리즘
- 매칭 점수 및 이유 제공


## 
1. 설문 데이터를 기반으로 사용자 프로필 생성
2. 각 항목별 가중치 적용하여 점수 계산
3. 갈등 요소에 대해 하드 컷오프 적용
4. 최종 매칭 점수 산출

##
클러스터링으로 룰 기반이 아니라 진짜 ai 사용해서 사용자에게 추천
1. 설문 데이터 수집 
2. 설문 데이터를 벡터 형태로 변환 (Embedding) 
3. 사용자 간 유사도 계산 (Cosine Similarity) 
4. 유사한 사용자 그룹화 (Clustering) 
5. 최종 매칭 추천


## 향후 가능성
- 실제 사용자 데이터 기반 개선
- 머신러닝 적용 (클러스터링 / 임베딩)
- 사용자 피드백 반영 시스템
